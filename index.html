HTML - Hyper Text Markup Language

You will learn:

  • Learn the anatomy of HTML syntax to structure your websites
  • Learn HTML doctypes & HTML5 Boilerplate well

  • Learn to use HTML tags or elements - headings, paragraphs

Week-03 & 04: CSS3 - Coloring Your Websites

How much CSS3 the students will learn in 3rd & 4th weeks?

We expect the students will continue leaning HTML5 and start learning CSS3 in next two weeks. They will start creating beautiful web pages using HTML5 & CSS3. Anyone can learn HTML5 & CSS3 in 4 weeks. However, 1st month can be still challenging for many. But "Nothing is impossible!

CSS3 - Cascading Style Sheets

What the students will learn:

  • Understand what are cascading style sheets
  • How easily the students can color and style websites
  • Understand CSS selectors and properties
  • Using inline, internal and external CSS
  • Learn and understand the CSS Box Modeling
  • Understand CSS specificity and inheritance
  • use developer tools to inspect and diagnose CSS
  • CSS positioning and display properties
  • font styling using CSS and web safe fonts
  • Implement responsiveness using CSS media queries
  • CSS float and clear properties
  • combine CSS selectors and understand selector priority
  • Understand CSS coding best practices
